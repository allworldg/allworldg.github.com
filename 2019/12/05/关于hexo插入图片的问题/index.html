关于hexo插入图片的问题

Dec 5, 2019

先放图，我的英雄学院

概述

之前写的博客都是纯文字的版本，所以阅读起来比较枯燥，而且对于问题的阐述和解决都不够有效率，所以今天阅读了一下markdown文档，学习了一下插入图片的几种方法。

方法

  1. ![alt 属性文本](图片地址)

    或者是

    ![alt 属性文本](图片地址 "可选标题")
    • 首先加一个！
    • 然后在[]里面添加图片的文字描述，在他人观看时不显示图片的时候可以显示出图片的文本解释
    • 在后面添加()，括号里填入图片的地址
  2. ![文本](图片地址)

    或者是

    ![文本](图片地址 "可选标题")

    第二种方法不会直接显示图片，而是显示指向图片的超链接。

需要注意的是，由于需要上传到博客网站上，所以图片最好不要要随意放在文章文件夹内或者是使用文件相对地址，容易导致博客的框架与其起冲突，很难管理，我们可以在GitHub上面新建一个仓库保存图片，在使用地址的时候将图片网页地址中的blob改成raw即可成为外链。
