Java学习:枚举笔记

Jan 21, 2020

看到枚举知识点，先简单总结一下：

  1. 枚举其实是一种类，与其他类采用class + 类名称不同，枚举的格式是enum + 类名称。
  2. 枚举是为了更简洁设置不同常量而去使用的，枚举的构造器是私有的，只有编译器能调用。
  3. 枚举的原理：暂时在编译后先通过反编译来查看，可以通过javap 类名或者是javap -c 类名

从图中可以清晰看到Enum实现的原理，需要注意的是，图中建立的成员变量都是静态且为final。
3. 比较可以直接用== 。

  1. Java.lang.Enum的方法
    1. String toString():返回枚举常量名字
    2. int compareto(E other):如果常量在other前面，则返回负数，负数大小表明常量在other前几个，相反正数表示在other后几个，0则表示相等。
